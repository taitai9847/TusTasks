# TusTasks
初学者が学ぶであろうことを詰め込んだタスク管理アプリです。
備忘録も含まれています。コードが参考になれば幸いです。

## 学べること
- R.swiftの使い方
- [Realm](https://realm.io/jp/docs/swift/latest/)
- 画面遷移
- vcでの値の受け渡し
- TableViewとカスタムセル

## 1. リポジトリをクローン
```
$git clone https://github.com/taitai9847/TusTasks.git
$cd Tustasks

//正しい階層にいるのかどうか確認してください
$pwd
```
## 2. Xcodeで開く
`TusTasks.xcworkspace`を開いてください。
